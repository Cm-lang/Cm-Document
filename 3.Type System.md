# bC 编程语言
## 三、类型系统

### 1.一类类型

一类类型指定了程序中主要的几种类型，bC语言有以下一类类型：

* func		函数
* const		值
* var		变量
* struct	结构
* enum		枚举

### 2.二类类型

#### (1) 整数类型
* i8		8位有符号整数
* i16		16位有符号整数
* i32		32位有符号整数
* i64		64位有符号整数
* u8		8位无符号整数
* u16		16位无符号整数
* u32		32位无符号整数
* u64		64位无符号整数

#### (2) 浮点数类型
* f32	32位浮点数类型
* f64	64位浮点数类型
