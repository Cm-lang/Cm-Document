# Cm编程语言

## 流程控制

### 分支

#### if语句

```bnf

if ::= <"if"><"("><条件表达式><")"><语句>
                [<"else"> <语句>]

```

例子：

```cm
var i = 0;
let x = true;
let y = false;

if(x) i++;
else if(y) i--;
else{
	i+=2;
}
```

#### if表达式
当if满足以下条件时，if可作为表达式使用：
* 拥有else
* if和else内的两块语句同时也是表达式
* if和else内的两块语句表达式返回值类型相同

例子：

```cm
let x = 1;
let y = if(x > 0) 1 else x + 5;
let z = if(x > 1){
	x + 5 > 0
}
else{
	false
}
```

### 循环

#### while循环

```bnf
while ::= <"while"><"("><条件表达式><")"><语句>
```

例子：

```cm
while(true)
{
     /*body*/
}
```

#### loop循环
```bnf
loop ::= <"loop"><语句>
```

例子：
```cm
loop{ /*body*/ }
```

### 打断循环

#### break
break可以从当前循环跳出。

#### continue
continue可以停止当前循环并且进入下一次循环。
